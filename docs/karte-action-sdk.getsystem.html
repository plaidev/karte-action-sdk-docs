getSystem() function #

KARTE の設定を取得する

Signature:

typescript
declare function getSystem(): SystemConfig;

Returns:

SystemConfig

現在の KARTE システムの設定を返します

Remarks #

アクションが表示されて設定されるため、onShow() フック関数で利用できます。 取得できる設定は、APIキー、接客ID、アクションショートIDなどです。
